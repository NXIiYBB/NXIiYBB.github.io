My Blog

Equipment Borrowing System for Laboratory Facilities Project: Challenges and Solutions

July 9, 2025

One of the most interesting projects I worked on was a demo web application for managing equipment borrowing in the CDLC room. I developed the system using Django, and it included separate interfaces for admins and users, as well as a request and approval process.

A major challenge I faced was designing a system that handled user roles effectively. Making sure admins had different access levels from regular users required a deeper understanding of Django’s authentication system. I also had to deal with database relationships, especially linking borrowed items to user records and approval statuses.

Overall, this project taught me how to turn an idea into a functional system and how to handle real-world development problems — both technically and in terms of user experience.
